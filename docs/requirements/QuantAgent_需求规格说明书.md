## 1. 背景与目标

### 1.1 业务背景

传统量化交易系统依赖人工编写策略规则，门槛高、迭代慢。用户需要根据自身操作风格（保守/激进/高频）自动生成专属量化策略，并能验证策略有效性。

### 1.2 解决什么问题

| 问题 | 解决 |
|------|------|
| 用户不知道适合什么策略 | AI 分析操作风格 → 生成匹配策略 |
| 策略效果无法验证 | 回测引擎验证策略表现 |
| 策略参数难以优化 | AI 参数调优 + 多周期回测 |
| 策略库难以维护 | 结构化策略管理 + 版本记录 |

### 1.3 目标

1. **操作风格分析** — 根据用户历史操作数据，输出风格画像（风险偏好/交易频率/持仓周期）
2. **策略 AI 生成** — 基于风格画像，AI 自动生成策略规则（指标/参数/仓位/止损）
3. **回测验证** — 对生成策略做多周期回测，输出绩效指标
4. **策略管理** — 策略库存储、版本管理、对比分析

## 3. 用户故事

| 角色 | 我想要... | 所以... |
|------|---------|--------|
| 散户 | 输入我的交易记录，系统告诉我是什么风格 | 得到风格画像 + 匹配策略 |
| 散户 | 系统自动生成适合我的策略 | 不用自己研究指标参数 |
| 散户 | 看到策略在历史数据上的表现 | 判断策略是否靠谱 |
| 散户 | 同一个策略，不同参数的表现对比 | 找到最优参数 |
| 量化研究员 | 管理多个策略版本 | 策略库 + 版本对比 |

## 4. 验收标准

| ID | 验收条件 | 验证方式 |
|----|---------|---------|
| V01 | 输入交易记录，返回风格画像 | 单元测试 |
| V02 | 相同风格输入，生成策略结果稳定 | 多次调用结果一致性 |
| V03 | 策略在历史数据上回测成功 | 回测引擎输出绩效报告 |
| V04 | 绩效指标计算正确（夏普/回撤/胜率）| 与已知基准对比 |
| V05 | 策略可保存到策略库 | CRUD 操作 |
| V06 | 策略可对比（两策略并排绩效）| 界面展示 |

## 5. 非功能需求

| 需求 | 说明 |
|------|------|
| **性能** | 单次策略生成 < 5s，回测 60 日数据 < 3s |
| **可扩展** | 新增市场/指标不影响核心逻辑 |
| **可测试** | 核心逻辑单元测试覆盖 ≥ 80% |
| **数据隔离** | 用户策略数据隔离，不泄露 |

## 6. 数据字典

### 6.1 用户风格画像

| 字段 | 类型 | 说明 |
|------|------|------|
| style | enum | 保守/稳健/平衡/积极/激进 |
| riskScore | float | 风险评分 0-100 |
| avgHoldDays | float | 平均持仓天数 |
| tradeFrequency | string | 高频/中频/低频 |
| maxDrawdownTolerance | float | 可接受最大回撤 % |

### 6.2 策略

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | UUID |
| name | string | 策略名称 |
| style | string | 对应风格 |
| rules | JSON | 策略规则（指标+参数+仓位+止损）|
| version | int | 版本号 |
| createdAt | timestamp | 创建时间 |

### 6.3 策略规则结构

```json
{
  "indicators": ["MA", "RSI", "MACD"],
  "params": {
    "MA_period": 20,
    "RSI_period": 14,
    "RSI_overbought": 70,
    "RSI_oversold": 30,
    "MACD_fast": 12,
    "MACD_slow": 26,
    "MACD_signal": 9
  },
  "entry": { "type": "cross", "condition": "MA_cross_RSI" },
  "exit": { "type": "stop_loss", "value": 0.05 },
  "position": { "type": "fixed", "value": 0.2 },
  "market": "A-share"
}
```

### 6.4 回测结果

| 字段 | 类型 | 说明 |
|------|------|------|
| totalReturn | float | 总收益率 |
| annualReturn | float | 年化收益率 |
| sharpeRatio | float | 夏普比率 |
| maxDrawdown | float | 最大回撤 % |
| winRate | float | 胜率 |
| profitLossRatio | float | 盈亏比 |
| totalTrades | int | 总交易次数 |

## 7. 已知限制

- V1 暂不支持实盘交易，只做策略生成 + 回测验证
- V1 数据源仅支持 CSV 文件导入（A股日线数据）
- AI 策略生成依赖 LLM API（需配置 API Key）
